Projects!

Here are two project that I have worked on during my time as an undergraduate student.


Reddit sentiment analisis

This group project was made for a data science class. It was designed to retrieve data from the reddit API and analyze the sentiment of each post. A simple web based interface was made in order to facilitate the use of the program. You can access it here: Reddit Sentiment Analysis. There you insert the name of a subreddit and it will analyze its dominant sentiment for you. This is the collaborative github github
and the collaborators were Jasiel Rivera, Michael Terrefortes, Jose Portela and myself.


Exploring the evolution of Lego with Data Camp

This datacamp project was made for a data science class. It was designed to make me more familiar with the python language. It also exposed us for the first time to the use of google collab and jupyter notebooks. In this projcts we used the pandas library to manipulate data sets with information about legos. This is the github github where you can see the completed jupyter notebook.
